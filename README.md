# Календарь2 
Приложение для управления своими задачами и составления расписаний.

# Установка
Чтобы собрать проект, вам понадобится `npm`, `electron-forge` и `vite`.
1) Скачайте папку с проектом (или клонируйте её с помощью `git clone`)
2) Перейдите в неё и пропишите следующие команды в консоль:
  `npm install`, это установит необходимые зависимости для сборки;
  `npm start`, запуск с помощью локального сервера.

\* Чтобы собрать из проекта `.exe` файл, пропишите в консоль:
  `npm run publish`, это создаст ярлык на рабочем столе с приложением.
